# Smart City Dashboard

A real-time smart city monitoring dashboard showing traffic, pollution, weather, and infrastructure data.

## Features
- Live traffic monitoring
- Air quality tracking
- Weather updates
- Energy consumption stats

## Tech Stack
- React
- Tailwind CSS
- Node.js

## How to Run
npm install
npm run dev
- Tailwind CSS
- shadcn-ui components

> No external images or branding are included by default. Remove any unused assets without affecting functionality.
